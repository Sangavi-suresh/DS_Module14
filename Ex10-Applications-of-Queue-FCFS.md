2.Read number of processes n
3.Read burst time for each process
4.Calculate waiting time:
5.First process → waiting time = 0
6.Others → sum of previous burst times
7.Calculate turnaround time:
8.TAT = Burst Time + Waiting Time
9.Display process details (BT, WT, TAT)
